TFS Colorizer Options

Use the settings below to customize the color of task cards on a TFS board.

  • Starts With - The task text starts with the supplied text.
  • Ends With - The task text ends with the supplied text.
  • Contains - The task text contains the supplied text.
  • RegEx - The task text matches the regular expression provided in the supplied text.
